Cream
===================


Cream is a jhipster 3 monilitic application based on **SpringBoot - AngularJS 1**[^stackedit]. 

----------


Requirement 
-------------------
JDK 8 <br/>
MysqlDB <br/>
maven >= 3.0<br/>
bower<br/>
grunt<br/>
 

To run 
---------

> **cmd :** bower install <br/>
> **cmd :** mvn clean install <br/>
> **cmd :** mvn <br/>
