Cream
===================


Cream is a jhipster 3 monilitic application based on **SpringBoot - AngularJS 1**[^stackedit]. 

----------


Requirement 
-------------------
JDK 8
MysqlDB 
maven >= 3.0
bower
grunt
 

To run 
-------------------
> **cmd :** bower install 
> **cmd :** mvn clean install 
> **cmd :** mvn
